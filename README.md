# 🎬 Netflix Clone

A modern Netflix clone built using React.js that replicates the UI and basic functionality of Netflix.

---

## 🚀 Features

* 🔥 Attractive Home Page UI
* 🎥 Movie Cards Section
* 🎯 Hero Banner with Background Image
* 📂 Category-wise Movie Lists
* 📱 Responsive Design

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 CSS
* ⚡ Vite

---

## 📁 Project Structure

```
src/
 ├── assets/
 ├── components/
 │    ├── Navbar/
 │    ├── Footer/
 │    ├── TitleCards/
 ├── pages/
 │    ├── Home/
 │    ├── Player/
 ├── App.jsx
 ├── main.jsx
```
